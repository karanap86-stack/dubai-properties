## üîÑ **Converting SVG to Other Formats**

### **Online Tools (Free):**
1. **CloudConvert.com** - SVG to PNG, JPG, PDF
2. **Inkscape** (software) - Professional editing + export
3. **SVGOMG.net** - Optimize/compress SVG files

### **Recommended Exports:**
- **PNG (Transparent):** 2000px width, 300 DPI (print quality)
- **JPG (White BG):** 1200px width (web use)
- **PDF:** Vector format (for printers/designers)

### **For Print (Business Cards, Brochures):**
- Export at **300 DPI minimum**
- Use PNG with transparent background OR PDF vector
- Provide CMYK version to printer (convert RGB ‚Üí CMYK in Photoshop)
