# Departments & Hierarchy

## Departments (industry standard)
- HR
- Sales
- Accounts/Finance
- Marketing
- Support/Customer Service
- Tech/IT
- Legal
- Operations
- Product
- Admin/Facilities
- Procurement
- R&D
- Compliance
- Analytics
- Training

## Hierarchy (for each department)
- Master Admin (full system access)
- Backup Master Admin
- Department Admin (full department access)
- Manager (team/department data)
- Agent/User (own/assigned data)
- AI Master Admin, AI Admin, AI Manager, AI Agent (mirrored for automation)
- Tech Admins (only for Tech/IT)

## Access Control
- Master Admin: all
- Department Admin: department
- Manager: team/department
- Agent/User: own/assigned
- Tech Admin: core code/settings
- Audit trail for all actions
